# OmniTrade ERP SaaS化规划方案

> **目标**：让无特殊要求的企业能够直接使用该系统，实现开箱即用的跨境电商ERP SaaS平台

---

## 📋 当前系统现状

### 现有架构
- ✅ 16个微服务（Spring Cloud Alibaba）
- ✅ 5大电商平台对接（Amazon、eBay、Shopee、Lazada、TikTok）
- ✅ AI智能功能（定价、预测、客服、描述生成）
- ✅ 完整前端页面（Vue 3）
- ✅ Docker + Kubernetes部署支持

### 存在的问题（转SaaS需要解决的）
1. **无多租户支持** - 所有企业共用一套数据库/服务
2. **缺乏租户隔离** - 数据完全混杂
3. **部署复杂** - 需要专业运维
4. **配置硬编码** - 难以适配不同企业
5. **无订阅计费** - 缺乏商业化能力

---

## 🎯 SaaS化战略目标

### 第一阶段：基础多租户架构（v2.0.0）
**目标**：实现核心多租户能力，支持多个企业共用系统

### 第二阶段：配置化与自助服务（v2.1.0）
**目标**：实现配置驱动，企业可自助开通使用

### 第三阶段：商业化与生态（v2.2.0）
**目标**：完整的SaaS商业模式，支持更多企业

### 核心组件

#### 1. 租户管理服务（Tenant Service）
```java
// 租户实体
public class Tenant {
    private String id;              // 租户ID
    private String name;            // 企业名称
    private String domain;          // 自定义域名
    private TenantPlan plan;        // 订阅计划
    private TenantStatus status;    // 租户状态
    private String region;          // 数据区域
    private Map<String, String> settings; // 租户配置
    private LocalDateTime expiresAt; // 到期时间
}

// 订阅计划
public enum TenantPlan {
    STARTER,    // 入门版 - 1000订单/月
    PROFESSIONAL, // 专业版 - 10000订单/月
    BUSINESS,   // 商业版 - 50000订单/月
    ENTERPRISE  // 企业版 - 无限制
}
```

#### 2. 多租户数据访问层
```java
// 租户上下文
public class TenantContext {
    private static final ThreadLocal<String> CURRENT_TENANT = new ThreadLocal<>();
    
    public static void setTenant(String tenantId) {
        CURRENT_TENANT.set(tenantId);
    }
    
    public static String getTenant() {
        return CURRENT_TENANT.get();
    }
}

// 多租户Repository
public interface MultiTenantRepository<T> {
    @Query("SELECT t FROM #{#entityName} t WHERE t.tenantId = :tenantId")
    List<T> findByTenantId(@Param("tenantId") String tenantId);
}
```

#### 3. 租户路由过滤器
```java
@Component
public class TenantFilter extends OncePerRequestFilter {
    @Override
    protected void doFilterInternal(HttpServletRequest request, 
                                    HttpServletResponse response,
                                    FilterChain chain) {
        // 从域名/Header/Token获取租户ID
        String tenantId = resolveTenantId(request);
        TenantContext.setTenant(tenantId);
        try {
            chain.doFilter(request, response);
        } finally {
            TenantContext.clear();
        }
    }
}
```
